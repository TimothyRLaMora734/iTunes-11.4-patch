# iTunes 11.4 patch
The iTunes 11.4 patch is a command line tool for running iTunes 11.4 on Catalina and older

# Usage
chmod +x ./iTunes\ 11.4\ patch.sh —> makes the script runnable  
sudo ./iTunes\ 11.4\ patch.sh —> runs the script
